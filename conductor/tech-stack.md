# Tech Stack

## Core
- **Language:** Ruby (>= 3.0.0)
- **CLI Framework:** Thor
- **Database:** SQLite3
- **Cloud Storage:** Google Cloud Storage (GCS)
- **AI/LLM:** Google Gemini (via direct REST API)

## Infrastructure & Tools
- **Dependency Management:** RubyGems & Bundler
- **Task Runner:** Rake
- **Testing:** Minitest
- **Version Control:** Git
